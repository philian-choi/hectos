# Product Brief: Hectos

## Executive Summary

**Hectos**는 hundredpushups.com의 검증된 6주 프로그램을 정확히 구현한 푸쉬업 트레이닝 앱입니다. 기존 앱들의 공통적인 문제점(부정확한 카운팅, 유연성 부족, 숨겨진 결제)을 모두 해결하여, 푸쉬업을 한 개도 못하는 초보자도 6주 만에 100개를 달성할 수 있도록 돕습니다.

**핵심 가치:** "정확하고, 유연하며, 정직한" 푸쉬업 트레이닝 경험

**비즈니스 모델:** 일회성 결제 - 한 번 구매로 100개 달성까지 완벽한 코칭 제공

**타겟 시장:** 글로벌 (한국어 + 영어 다국어 지원)

---

## Core Vision

### Problem Statement

푸쉬업은 가장 효과적인 상체 운동 중 하나이지만, 대부분의 사람들이 10개도 채 못하고 포기합니다. 시중의 푸쉬업 앱들은 이 문제를 해결하겠다고 약속하지만, 오히려 사용자를 좌절시키고 있습니다.

### Problem Impact

- **카운팅 부정확**: 반복을 놓치거나 중복 카운트하여 동기 저하
- **유연성 부족**: 세트 완료 실패 시 조정 불가, 사용자 능력에 맞추지 못함
- **숨겨진 결제**: "무료"로 시작했다가 주요 기능에서 결제 요구
- **비현실적 진행**: 사용자 상태를 무시한 고정된 플랜으로 부상/좌절 유발

### Why Existing Solutions Fall Short

- hundredpushups.com 프로그램을 정확히 구현한 앱 부재
- 대부분 이름만 "100 pushups"이고 자체 프로그램 사용
- Column 시스템, Exhaustion Test, 적응형 진행 미구현
- 초보자 친화적이지 않은 UX

### Proposed Solution

**Hectos**는 다음을 제공합니다:

1. **정확한 카운팅** - 터치 기반 카운터 (가장 신뢰할 수 있는 방식)
2. **검증된 프로그램** - hundredpushups.com 6주 프로그램 완벽 구현
3. **적응형 플래닝** - 실패해도 자동 조정, 주차 반복 지원
4. **정직한 가격** - 일회성 결제로 100개 달성까지 완전 코칭
5. **초보자 친화적** - 0개부터 시작 가능, 격려와 동기부여

### Key Differentiators

| 차별점 | Hectos | 경쟁 앱 |
|--------|--------|---------|
| 프로그램 | hundredpushups.com 공식 | 자체 프로그램 |
| 결제 모델 | 일회성, 투명 | 숨겨진 구독/IAP |
| 적응성 | 실패 시 자동 조정 | 고정 플랜 |
| 타겟 | 초보자 중심 | 애매한 타겟 |
| 글로벌 | 다국어 지원 | 대부분 영어 |

## Target Users

### Primary User: 푸쉬업 초보자

**페르소나: "민수"**

| 항목 | 내용 |
|------|------|
| **프로필** | 28세, 사무직 직장인 |
| **신체 상황** | 책상에 오래 앉아 어깨가 뻣뻣하고 팔 힘이 약함 |
| **현재 능력** | 푸쉬업 5개도 힘겨움 |
| **동기** | "100개 하는 사람들이 멋있어 보여서 나도 해보고 싶다" |
| **좌절 경험** | 유튜브 보고 따라했다가 바로 포기, 앱 깔았는데 중간에 결제 요구해서 삭제 |
| **성공 비전** | 6주 후 친구들 앞에서 100개 연속 완료! |

**특징:**
- 운동 습관이 없거나 오래 쉬었던 사람
- 푸쉬업 0-10개 수준
- 명확한 목표와 기간이 있으면 동기부여됨
- 실패에 민감 - 격려와 적응형 플랜 필요

### Secondary Users

현재 버전에서는 Primary User에 집중합니다. 향후 확장 고려:
- 체력 측정이 필요한 군인/경찰 준비생
- 체육 수업에서 활용하는 교사
- 무릎 푸쉬업부터 시작하는 초보 여성

### User Journey

```
1. 발견 → 앱스토어에서 "100 pushups" 검색
2. 설치 → 무료 다운로드, 간단한 온보딩
3. 초기 테스트 → 현재 몇 개 할 수 있는지 측정
4. 레벨 배정 → Column 1/2/3 중 적합한 레벨 자동 배정
5. 유료 전환 → 6주 프로그램 일회성 구매
6. 매일 운동 → 주 3회, 5세트씩 진행
7. 중간 테스트 → 2주/4주차 Exhaustion Test로 레벨 재조정
8. 🎉 성공 → 6주 후 100개 달성, 축하 화면!
```

### Key Performance Indicators

1. **North Star Metric:** "100개 달성 사용자 수"
   - 사용자와 비즈니스 모두에게 의미있는 핵심 지표
   - 앱의 존재 이유를 직접 측정

2. **Leading Indicator:** "2주차 완료율"
   - 초기 이탈 예측 지표
   - 온보딩 및 초기 경험 품질 반영

3. **Retention Metric:** "프로그램 중 운동 완료 비율"
   - 주 3회 운동 중 실제 완료 비율
   - 사용자 참여도 및 프로그램 적합성 측정

## MVP Scope

### Core Features (MVP 필수 기능)

| # | 기능 | 설명 |
|---|------|------|
| 1 | **초기 테스트** | 사용자 최대 푸쉬업 개수 측정 → Column 자동 배정 |
| 2 | **6주 프로그램** | hundredpushups.com 전체 6주 구조 완벽 구현 |
| 3 | **운동 세션** | 세트/반복 표시, 휴식 타이머, 터치 카운터 |
| 4 | **Exhaustion Test** | 2주/4주차 중간 테스트로 레벨 재조정 |
| 5 | **적응형 플래닝** | 실패 시 자동 조정, 주차 반복 지원 |
| 6 | **진도 추적** | 완료 세션, 현재 주차/일차, 총 푸쉬업 수 |
| 7 | **IAP (일회성)** | 6주 프로그램 일회성 구매 |
| 8 | **다국어 (한/영)** | 한국어 + 영어 지원 |

### Out of Scope for MVP

| 기능 | 제외 이유 | 향후 버전 |
|------|----------|----------|
| AI 폼 분석 | 복잡도 높음, 카메라 권한 필요 | v2.0 |
| 소셜/리더보드 | 백엔드 인프라 필요 | v2.0 |
| Apple Watch 연동 | 추가 플랫폼 개발 필요 | v1.5 |
| 다른 운동 프로그램 | MVP 범위 제한 | v2.0 |
| 근접 센서 카운팅 | 터치가 더 신뢰성 높음 | v1.5 (옵션) |
| 클라우드 동기화 | 백엔드 인프라 필요 | v2.0 |
| 푸시 알림 | 복잡도 대비 가치 낮음 | v1.5 |

### MVP Success Criteria

**MVP 성공 기준:**
1. **기술 검증**: 앱스토어 승인 및 크래시 없이 안정 동작
2. **사용자 검증**: 최소 100명 프로그램 시작, 2주차 완료율 50% 이상
3. **비즈니스 검증**: 첫 달 구매 전환율 3% 이상
4. **피드백 수집**: 앱스토어 리뷰 10개 이상, 평점 4.0★ 이상

**Go/No-Go 결정 기준:**
- 위 4가지 중 3개 이상 충족 시 → v1.5 개발 진행
- 2개 이하 충족 시 → 피봇 또는 핵심 문제 해결 후 재시도
