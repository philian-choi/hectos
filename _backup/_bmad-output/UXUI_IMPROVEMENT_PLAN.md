## 📊 Part 1: Nike Run Club 분석

### 1.1 디자인 시스템 핵심 요소

| 요소 | Nike Run Club 스펙 | Hectos 현재 | 개선 방향 |
|-----|------------------|------------|---------|
| **배경색** | 순수 블랙 (#000000) | #0F172A (네이비) | 유지 (더 부드러운 느낌) |
| **Primary 컬러** | 네온 Volt 그린 (#CDFF00) | 주황 (#FFB800) | 유지 (차별화) |
| **타이포그래피** | Futura Bold Condensed + 가벼운 본문 | Inter | 유지 |
| **레이아웃** | 미니멀, 여백 풍부 | 적절 | 여백 더 확대 |

### 1.2 Nike Run Club UI 패턴

#### 홈 화면 구조
```
┌─────────────────────────────────────┐
│ [로고]                    [프로필]  │
├─────────────────────────────────────┤
│                                     │
│        크고 간결한 제목             │
│        (예: "Today's Run")          │
│                                     │
├─────────────────────────────────────┤
│    ┌─────────────────────────┐     │
│    │                         │     │
│    │    핵심 CTA 버튼        │     │
│    │    (GO / START)         │     │
│    │                         │     │
│    └─────────────────────────┘     │
├─────────────────────────────────────┤
│  통계 카드 1    │   통계 카드 2     │
│  (이번 주)      │   (누적)          │
└─────────────────────────────────────┘
```

#### 색상 코드 시스템
- **Volt Green (#CDFF00)**: 메인 CTA 버튼
- **White**: 주요 텍스트
- **Gray 계열**: 보조 텍스트, 비활성 상태
- **Gradient (초록→노랑→주황→빨강)**: 속도/성능 표시

#### 뱃지 시스템
- **월간 마일리지**: Bronze < Silver < Gold < Platinum
- **개인 기록(PR)**: Furthest Run, Fastest 5K 등
- **히든 뱃지**: 특정 날짜 운동 (발렌타인, 새해 등)

| 트리거 | 애니메이션 | 적용 위치 |

| 원칙 | 설명 | Hectos 적용 |

## 🎨 Part 4: 색상 시스템 업데이트

### Nike/Apple 스타일 적용

```typescript
// constants/theme.ts 업데이트

export const colors = {
  // 기존 Primary 유지 (차별화)
  primary: '#FFB800',      // Hectos 주황
  primaryLight: '#FFCC4D',
  primaryGlow: 'rgba(255, 184, 0, 0.3)',
  
  // Apple Activity Ring 스타일 추가
  rings: {
    move: '#FF2D55',       // Apple 빨강
    exercise: '#30D158',   // Apple 초록  
    stand: '#0A84FF',      // Apple 파랑
  },
  
  // Nike 스타일 액센트
  accent: {
    volt: '#CDFF00',       // Nike Volt (선택적)
    fire: '#FF6B35',       // 스트릭 불꽃
    gold: '#FFD700',       // 성취 뱃지
  },
  
  // 성능 그라디언트 (Nike 스타일)
  performance: {
    excellent: '#30D158',  // 초록
    good: '#CDFF00',       // 연두
    average: '#FFB800',    // 주황
    slow: '#FF3B30',       // 빨강
  }
};
```
