## VP 상세 설명

### VP-T1: "0개부터 100개까지. 6주면 충분합니다."
- **접근법**: Benefit → Value
- **고객 언어**: "I want to go from 0 to 100"
- **소구 방향**: 명확한 목표, 달성 가능성 강조

### VP-T2: "초급/중급/상급. 당신에게 맞는 곳에서 시작."
- **접근법**: Painpoint → Solution
- **고객 언어**: "Unrealistic training plans"
- **소구 방향**: 무리하지 않는 현실적 시작

### VP-T3: "오늘 뭘 해야 하는지, 앱이 알려줍니다."
- **접근법**: Painpoint → Solution
- **고객 언어**: "Just tell me what to do each day"
- **소구 방향**: 의사결정 피로 해소

### VP-T4: "정체기? 과학적 루틴이 뚫어드립니다."
- **접근법**: Painpoint → Solution
- **고객 언어**: "I plateaued at week 4"
- **소구 방향**: 문제 해결 보장

### VP-T5: "배지, 알림, 축하 효과. 혼자여도 포기 없이."
- **접근법**: Benefit → Value
- **고객 언어**: "Community challenges provide supportive environment"
- **소구 방향**: 게이미피케이션으로 리텐션
