# Hectos Personas Overview

Hectos의 성공적인 LMF를 위해, 리서치 데이터에서 발견된 주요 **문제 상황(Painpoint)**을 중심으로 4가지 페르소나를 정의했습니다.

| ID | Name | Core Problem (Painpoint) | Key Motivation (Why Hectos?) | Status |
|----|------|--------------------------|------------------------------|--------|
| **01** | **Kevin** | **인지적 부하** (숫자 세기 헷갈림) | 터치 없는 정확한 자동 카운팅 | Primary |
| **02** | **Sarah** | **프라이버시 불신** (카메라 노출 우려) | 온디바이스 AI (서버 전송 없음) | Secondary |

| **04** | **Elena** | **투박한 UX** (동기부여 저하) | 프리미엄 Glassmorphism UI | Secondary |

우리는 우선적으로 **Kevin(숫자 세기 귀찮음)**과 **Travis(정체기 극복)**를 핵심 퍼소나로 설정합니다. 이들은 푸쉬업 실력을 늘리고 싶은 본연의 욕구가 가장 강하며, Hectos의 기능적 정체성과 가장 잘 맞닿아 있습니다.

- **Secondary Persona**: Sarah, Elena
